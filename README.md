#### Si quieres crear un nuevo proyecto en blanco usando NDK con svelte

1. pnpm create svelte@latest svelte-ndk-nostr
2. pnpm add @nostr-dev-kit/ndk
3. pnpm i
4. pnpm run dev
5. instalar extension de 'Svelte for VScode'
